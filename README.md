# Корпоративная Система Управления (MVP) 🚀

Асинхронное веб-приложение на **FastAPI** для управления сотрудниками, командами, задачами и рабочими встречами.

## ✨ Основные возможности
- **Управление задачами:** Создание, назначение исполнителей, отслеживание статусов и дедлайнов.
- **Командная работа:** Система кодов приглашений для вступления в команды.
- **Умное планирование встреч:** Автоматическая проверка накладок (overlap) в расписании сотрудников.
- **Система оценок:** Руководители могут оценивать выполнение задач после их завершения.
- **Интерактивная админка:** Полноценный интерфейс управления данными через браузер.
- **Контейнеризация:** Полная готовность к развертыванию через Docker.

## 🛠 Технологический стек
- **Framework:** FastAPI (Python 3.13)
- **Database:** PostgreSQL + SQLAlchemy 2.0 (Async)
- **Migrations:** Alembic
- **Admin Panel:** SQLAdmin (Jinja2 templates)
- **Auth:** FastAPI Users (JWT Strategy)
- **Dependency Management:** Poetry
- **Testing:** Pytest (Coverage > 70%)

## 🚀 Быстрый запуск через Docker

Убедитесь, что у вас установлены [Docker](https://www.docker.com/) и [Docker Compose](https://docs.docker.com/compose/).

1. **Клонируйте репозиторий:**
``` Bash
   git clone <url_вашего_репозитория>
   cd EffectiveMobile_FinProj
```
2. Подготовьте файл окружения:
    Скопируйте шаблон и при необходимости отредактируйте значения:

``` Bash
    cp .env.example .env
    cp .env.db.example .env.db
```

3. Запустите систему:

``` Bash
    docker compose up --build
```

После запуска приложение будет доступно по адресу: http://localhost:8000

## 🔗 Навигация по сервису
- Swagger UI (Документация API): http://localhost:8000/docs
- Админ-панель: http://localhost:8000/admin
- ReDoc: http://localhost:8000/redoc

## 🧪 Тестирование
В проекте реализован сложный интеграционный сценарий (test_full_business_flow), покрывающий весь цикл: регистрация, создание команды, постановка задачи, её выполнение и финальная оценка.

Для запуска тестов локально (требуется Poetry):

``` Bash
    poetry install
    poetry run pytest -v --cov=src
    Текущее покрытие кода: 73% ✅
```

- src/models/ — Описание моделей SQLAlchemy.
- src/schemas/ — Схемы валидации Pydantic.
